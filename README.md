# Ultimate Tic-Tac-Toe — Browser Game + AI

Jeu statique (HTML/CSS/JS) compatible **GitHub Pages**.

✅ Modes : Humain vs IA / Humain vs Humain / IA vs IA  
✅ Undo / Redo (propre)  
✅ Historique des coups  
✅ Surbrillance du dernier coup  
✅ Indication des plateaux FORCÉ/LIBRE  
✅ IA : Minimax (iterative deepening + transposition table) ou MCTS  
✅ Stats IA (temps, profondeur, nœuds, cache)  
✅ Raccourcis : U / R / N  
✅ Tests + CI (Vitest + GitHub Actions)
